Remote Development Guide
Your project is now synchronized with GitHub. Here is how you can work on it from anywhere.

Repository Info
URL: https://github.com/kjd1374/home-hub-gate
Branch: main
How to Work form Another Machine (e.g., Laptop)
1. First Time Setup (Clone)
On your external machine (laptop), run this command in your terminal to download the code:

bash
git clone https://github.com/kjd1374/home-hub-gate.git
cd home-hub-gate
npm install
2. Making Changes
After editing files on your laptop:

bash
git add .
git commit -m "Description of changes"
git push origin main
3. Syncing Changes Back to Server (Mac Mini)
When you return to your Mac Mini, or to update the server:

bash
cd /Users/jungdookim/NAS/home-hub
git pull origin main
# If you installed new packages:
npm install
# Restart server if needed
